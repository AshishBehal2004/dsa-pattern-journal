**Pattern Name: HashMap + Stack**

**Approach**
store each bracket in hashmap as key value(open as key, close as value)
Loop over the string chars
	  check if the current element is an open bracket
		 if yes then add to stack
		
	  otherwise check if the element is close bracket
		  check if stack is not emepty and the current element matches the top element of the stack
			  pop/remove from the stack
		  otherwise return false as stack can be empty or the element doesn’t match with top element of the stack
	  otherwise return false
	
	now check if stack is empty as by the time loop ends, all elements should be matched and popped out from the stack
	    return true
			
	otherwise return false
	
**TC: O(n)**
**SC: O(n) using stack and map**

class Solution {
    public boolean isValid(String s) {
        HashMap<Character, Character> map = new HashMap<>();
        Stack<Character> stack = new Stack<>();

        map.put('(',')');
        map.put('[',']');
        map.put('{','}');

        for (int i = 0; i < s.length();i++){
            if (map.containsKey(s.charAt(i))){ // checking if open bracket
                stack.push(s.charAt(i));
            }
            else if(map.containsValue(s.charAt(i))) {
                if (!stack.empty() && s.charAt(i) == map.get(stack.peek()) ){
                    stack.pop();
                }
                else{
                    return false;
                }
            }
            else{
                return false;
            }
        }
        if (stack.empty()){
            return true;
        }
        else{
            return false;
        }

    }
}
